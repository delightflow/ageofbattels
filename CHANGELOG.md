## v10 (2026-02-19)
- **모바일 UI 개선**: 사이드바 하단 버튼 항상 보이게 수정

  - ⚡ 낙뢰: 랜덤 위치에 범위 데미지 + 번개 이펙트
  - 🔥 전투 함성: 한 팀 공격력 1.5배 (5초)
  - 🏳️ 원군 도착: 랜덤 팀에 검병 2~3명 추가
  - 🌍 지진: 모든 유닛 이동속도 50% 감속 (4초)
  - 화면 상단에 이벤트 배너 표시 (페이드아웃)
